# tools

Вспомогательные скрипты v.3 (ingest, healthcheck, миграции).

| Скрипт | Назначение |
|--------|------------|
| `ingest_from_g.py` | Загрузка `.md`/`.txt`/`.pdf` с G: → Chroma (`--reset`, `--json`, `--source`) |
| `dzen_playwright_export.py` | Статья Дзен (/a/...) → `.md` через браузер |
| `dzen_scrape_probe.py` | Проверка: почему requests не работает |

Не дублировать `AIKIVAVIORA/tools/` — при необходимости копировать логику с согласия пользователя.
