# Локальная папка для ingest (опционально)

Положите сюда `.pdf`, `.txt`, `.md` для индексации **на своей машине**.

Содержимое **не коммитится** в git (см. корневой `.gitignore`).

В `.env` можно указать другой путь:

```env
RAG_SOURCE_PATH=data/rag_source
```

Или абсолютный путь к своей папке с материалами.
