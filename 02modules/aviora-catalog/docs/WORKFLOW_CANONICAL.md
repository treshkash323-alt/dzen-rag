# Главный файл (★) — Aviora Catalog

## Зачем

В `Python_kash` часто лежат **несколько копий** одного и того же текста (разные папки, архив, LM Studio). Каталог после **Scan** строит группы по хешу текста и предлагает **главный** путь (эвристика: `02modules/`, `cursor/`, без `archive`).

## Шаги

1. **Scan** (лента → Сервис) — дождаться 100% и тоста с числом групп дубликатов.
2. **Поиск** — ввести слово, **Enter**. Строки с **★ главный** идут выше; у копий — «копия (N)».
3. Открыть нужный файл → **Сервис → ★ Главный** — закрепить свою метку (перебивает авто).
4. Повторное нажатие **Главный** снимает метку (если вы сами пометили).
5. **ПКМ** по строке в поиске — открыть файл и сразу переключить метку.

## Где хранится

`05data/aviora_canonical.json` (рядом с модулем, не в git). **Сброс** сессии очищает и индекс, и реестр ★.

## Чат

В блоке «Индекс» для LLM строки с **★** — рекомендуемый файл в группе; список копий добавляется под главным.

## Сохранение отчётов (v0.3.10)

- После **поиска** (Enter) — кнопка **💾 Поиск** слева или лента **Сервис → Поиск MD**.
- После **ответа чата** — **💾 Ответ** справа или **Чат MD**.
- Файлы: `05data/aviora_search_saves/search_*.md` и `chat_*.md` в корне `Python_kash`.
- **Сервис → Отчёты** — открыть папку в дереве.
