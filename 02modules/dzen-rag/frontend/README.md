# Frontend — Dzen RAG

Чат по материалам канала. Без сборки — статические файлы, отдаёт backend.

## Запуск

1. Запустите API из `backend`:
   ```powershell
   cd backend
   .\run.ps1
   ```
2. Откройте в браузере: **http://127.0.0.1:8001/ui/**  
   (корень http://127.0.0.1:8001/ перенаправит на `/ui/`)

## Возможности

- Вопрос к RAG
- Переключатель LLM: auto / deepseek / lm_studio / none
- Список фрагментов-источников с score
- Статус индекса (чанки, DeepSeek)
