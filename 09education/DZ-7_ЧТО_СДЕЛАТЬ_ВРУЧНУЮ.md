## Шаг 1. Убедиться, что RAG живой (15 мин)

1. Запустите API: двойной клик `02modules\dzen-rag\start-api.bat`  
   (или `backend\run.ps1` из папки backend).
2. Откройте http://127.0.0.1:8001/ui/ — в шапке бейдж **«126 чанков»** (или > 0).  
   Если 0 — в `.env` укажите свой `RAG_SOURCE_PATH` на папку модуля 1.3 и выполните:
   ```powershell
   cd <корень AIKIVAVIORA_v.3_Cursor>
   python tools\ingest_from_g.py --reset
   ```
3. В **чате** задайте 3–4 вопроса по модулю 1.3 (список в ПЗ, раздел 5).
4. В **Swagger** http://127.0.0.1:8001/docs → POST `/chat`:
   ```json
   { "question": "Как приготовить борщ?", "provider": "auto", "top_k": 5 }
   ```
   Ожидание: ответ в духе **«Информация не найдена в базе знаний»** (или нет релевантных источников).
5. (Желательно) На `/ui/` загрузите тестовый PDF/TXT в блок **«Пополнение базы»** — статус «Загружено», счётчик чанков вырос.
